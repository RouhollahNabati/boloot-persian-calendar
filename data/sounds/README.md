# Adhan audio files

Bundled adhan presets for BOLOOT Persian Calendar:

| File | Preset | Description |
|------|--------|-------------|
| `mansouri.ogg` | منصوری | کریم منصوری (default) |
| `makkah.ogg` | مأذون حرم مکی | Adhan from Masjid al-Haram, Makkah |
| `madinah.ogg` | مأذون حرم مدنی | Adhan from Masjid an-Nabawi, Madinah |

## Attribution

### `mansouri.ogg`

- **Reciter:** کریم منصوری (Karim Mansouri)
- **Source:** [شهید آوینی — آرشیو اذان](http://dl.aviny.com/voice/azan/moazenin/47-karim-Mansori.mp3)
- **Format:** converted from MP3 to OGG Vorbis for in-app playback

### `makkah.ogg`

- **Source:** [PrayTimes.org — Adhan Makkah](https://praytimes.org/audio/sunni/Adhan-Makkah.mp3)
- **Format:** converted from MP3 to OGG Vorbis

### `madinah.ogg`

- **Source:** [PrayTimes.org — Adhan Madinah](https://praytimes.org/audio/sunni/Adhan-Madinah.mp3)
- **Format:** converted from MP3 to OGG Vorbis

## Custom adhan

Users can choose **فایل سفارشی** (custom) in settings and select any local
`.ogg`, `.wav`, `.mp3`, or `.flac` file.

## Install path

Installed to `/usr/share/boloot-calendar/data/sounds/` (or `$BOLOOT_DATA_DIR/sounds`).
